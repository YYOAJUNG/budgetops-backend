## 📖 문서 작성 규칙

새로운 문서를 작성할 때는 다음 규칙을 따라주세요:

1. **파일명**: 대문자와 언더스코어 사용 (예: `TESTING_GUIDE.md`)
2. **언어**: 한글 우선, 필요시 영어 혼용
3. **구조**: 
   - 목차 포함
   - 명확한 섹션 구분
   - 코드 예시 포함
   - 업데이트 이력 기록
4. **위치**: 모든 문서는 `docs/` 폴더에 보관

## 🔄 문서 업데이트

문서를 업데이트한 후에는 해당 문서의 **업데이트 이력** 섹션에 변경 내용을 기록해주세요.

**예시:**
```markdown
## 업데이트 이력

- 2024-12-08: 초기 문서 작성
- 2024-12-10: 새로운 테스트 패턴 추가
```
